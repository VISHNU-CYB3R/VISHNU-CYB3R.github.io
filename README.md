# Personal Portfolio 🚀

Welcome to the repository for my personal portfolio website! This is a clean, static website built from scratch to showcase my technical skills, academic journey, and professional projects. 

## 👨‍💻 About Me

I am **Vishnu C**, a BSc Computer Science student specializing in **Cybersecurity**. I have a strong foundation in ethical hacking, network security, and database management. My technical toolkit includes programming languages like **C, Python, and Java**, alongside data visualization tools like **Power BI**. 

When I'm not building web-based vulnerability scanning dashboards or analyzing market trends for my Telegram channel, *The Loot Radar*, I am a dedicated Veena player—a practice that helps me cultivate the discipline, patience, and creativity I bring to my technical work.

## ✨ Features

* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
* **About Section:** A brief overview of my background in computer science and cybersecurity.
* **Skills Showcase:** Highlighting my proficiency in networking, programming, and database management.
* **Projects Gallery:** Detailed cards for my current and past work, including my web-based vulnerability scanning dashboard.
* **Contact Form:** A simple, intuitive way for visitors and potential collaborators to get in touch.

## 🛠️ Tech Stack

This project is intentionally kept lightweight and fast, utilizing core web technologies:

* **HTML5:** For semantic structure and accessibility.
* **CSS3:** For styling, layout (Flexbox/Grid), and smooth animations.
* **JavaScript (Vanilla):** For interactive elements and DOM manipulation.
* **Formspree:** Integrated as a form backend service to handle and route messages from the Contact section directly to my email.
